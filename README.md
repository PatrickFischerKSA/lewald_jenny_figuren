- Dilemmata & Entscheidungen
- Perspektivwechsel, Identifikation, Fremderfahrung

## Struktur
- `index.html`
- `style.css`
- `app.js`
- `assets/` (optional für Bilder/Audio)

## GitHub Pages
1. Repo erstellen (oder in bestehendes Repo kopieren).
2. Dateien ins Repo-Root legen.
3. GitHub → Settings → Pages → Branch `main` / Folder `/root`.
4. Seite öffnen.

## Hinweise
- Fortschritt wird lokal im Browser gespeichert (localStorage).
- Export erzeugt eine JSON-Datei mit Antworten/Fortschritt.
- „Druck/PDF“ nutzt die Browser-Druckfunktion (als PDF speichern).
